Indian Restaurants Analysis

Cloud Assignment-2

Data Collection

The zomato was collected from kaggle and dataworld.com, this was used to make our predictions and visualisations

ETL(Extract Transform Load)

PIG Latin was used for removing the unnecessary data from dataset. Data merging and preprocessing was done in this stage.

Model Creation

Multiple linear regression model was developed for average price prediction and testing.

Visualisation

Tableau was used for our visualisation and dashboard analysis. Google Firebase is used for hosting this visualisation.
